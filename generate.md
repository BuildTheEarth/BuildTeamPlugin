# Generator

## Houses

To create a random house, just enter:
```
/gen house
```

### Wall Color:
To specify the wall color, enter:
```
/gen house -w <id>
```
To find the correct block id, click [here](https://www.digminecraft.com/lists/item_id_list_pc_1_12.php).

### Roof Color:
To specify the roof color, enter:
```
/gen house -r <id>
```
To find the correct block id, click [here](https://www.digminecraft.com/lists/item_id_list_pc_1_12.php).
